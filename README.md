# TextEmotionML

**TextEmotionML** es un proyecto de clasificación automática de emociones en texto.  
Su objetivo es construir un sistema capaz de identificar la emoción predominante en frases o mensajes escritos, permitiendo analizar de manera automatizada el contenido emocional presente en textos.

El proyecto aborda tareas fundamentales del procesamiento de lenguaje natural (NLP), como el preprocesamiento de textos, la representación de documentos y la construcción de un modelo capaz de reconocer patrones emocionales. El sistema busca clasificar textos en categorías como alegría, tristeza, enojo, miedo, sorpresa, entre otras, dependiendo del conjunto de datos utilizado.

El enfoque principal es desarrollar un modelo funcional, evaluar su desempeño con métricas estándar de clasificación y analizar su rendimiento para determinar qué tan bien puede detectar emociones en distintos tipos de textos.
